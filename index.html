Sunbiz Modernization News & Updates


Sunbiz Data Downloads are Changing

September 30, 2021

Updates to Sunbiz will include updated data schema (data file structure) and download procedures for Business Entity data files available from the Florida Department of State, Division of Corporations.   Read more   

Data downloads for other entity types will be described in future documents, which will be made available on this preview site. Click the button below to view instructions for accessing Business Entity data file downloads and to see more information on the upcoming new data file formats.


User Accounts Will Be Required for Filing Annual Reports

September 30, 2021

The updated Sunbiz system will require to create a user account with a unique username and a strong password to perform business filings, download data files, and file an Annual Report.   Read more   

With the current Sunbiz.org system, users are not required to provide a username and password for business filings and some data downloads. The modernized Sunbiz site, however, will require users to create individual user accounts, each with a unique username and strong password for secure authentication. User accounts will also be required for filing an Annual Report. Also, users that want to file an Annual Report that makes changes to their business entity will be required to have filing authority. Please click the button below for more information.

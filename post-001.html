Sunbiz Modernization News & Updates: Data Downloads


Sunbiz Data Downloads are Changing

April 30, 2021

Updates to Sunbiz will include updated data schema (data file structure) and download procedures for Business Entity data files available from the Florida Department of State, Division of Corporations. Data downloads for other entity types will be described in future documents, which will be made available on this preview site. Click the button below to view instructions for accessing Business Entity data file downloads and to see more information on the upcoming new data file formats.
